##########
flake8-csv
##########

A Flake8 plugin to generate error reports in CSV format.

Installation
============

Install from PyPI:

.. code-block:: bash

    $ pip install flake8-csv

Or from Git:

.. code-block:: bash

    $ pip install -e git+git://github.com/bennylope/flake8-csv.git#egg=flake8_csv

Usage
=====

Run Flake8 with the format option `csv`::

    flake8 --format=csv
